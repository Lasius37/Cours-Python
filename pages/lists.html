    Créer une liste

    • On peut créer une liste vide pour la compléter par la suite ; on utilise les [] : 
      liste_vide = []
    • On peut aussi créer une liste qui contient déjà des éléments : 
      ingredients = ["farine", "sucre", "lait"]

    Ajouter un élément dans une liste

    Le nouvel élément est toujours ajouté à la fin de la liste.

    ingredients.append("oeufs")

    Trouver un élément dans une liste

    • Pour récupérer la position d'un élément dans une liste (son index) : 
      ingredients = ["farine", "sucre", "lait", "oeufs"]
numero = ingredients.index("sucre")
# les index commencent à 0, donc le "sucre" est à l'index 1 : les index sont 0 ; 1 ; 2 ; 3. Ici, numero prendra la valeur 1.
    • Pour récupérer le 3ème élément dans la liste : 
      element = ingredients[2]
# les index commencent à 0. Ici, element prendra la valeur "lait"

    Supprimer un élément d'une liste

    • Ce code efface complètement l'élément sélectionné: 
      ingredients = ["farine", "sucre", "lait", "oeufs"]
ingredients.remove("lait")
    • Ce code permet de stocker l'élément supprimé dans une variable : 
      ingredients = ["farine", "sucre", "oeufs"]
element = ingredients.pop(1)  # dans notre cas, element sera égal à sucre

    Connaître la longueur d'une liste

    Cela peut être utile de connaître le nombre d'éléments dans une liste.

    ingredients = ["farine", "sucre", "oeufs"]
len(ingredients)  # renvoie 3

    Trier une liste

    Dans une liste de nombre, le tri est croissant. Dans une liste de chaînes, le tri est alphabétique.

    ingredients.sort()

    Parcourir une liste

    On réalise une boucle sur la liste, et chaque élément de la liste sera traité indépendamment.

    for element in ingredients:  # pour chaque élément de la liste "ingredients"
    print(element)  # affiche l'élément

    Erreurs fréquentes

    • Oubli des [] dans une liste : SyntaxError.
    • Oubli des , de séparation : SyntaxError ou comportement anormal.
    • Utilisation d'un index trop grand : IndexError.
