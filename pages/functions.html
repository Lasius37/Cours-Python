    Définir une fonction simple

    Les fonctions ont pour but de réaliser une série de tâches. On les nomme en snake_case.

    def dire_bonjour():  # initialisation de la fonction
    print("Bonjour !")

    Définir une fonction avec des paramètres

    La fonction réalise une série de tâches, mais qui dépendent de paramètres qu'on lui donne.

    def dire_bonjour_nom(nom):  # nom est un paramètre à donner à la fonction
    print(f"Bonjour, {nom} !")

    Récupérer le résultat d'une fonction

    Quand on veut récupérer le résultat d'une fonction dans une variable, on utilise le mot return.

    def test_temperature(valeur):
    if valeur <= 0:
        message =  "il gèle !"
    else:
        message = "il ne gèle pas"
    return message  # la fonction renvoie une valeur grâce à return

    Appeler une fonction

    Nous avons déjà appelé des fonctions : input(), print(), type()... sont des fonctions. Les trois fonctions ci-dessus sont appelées successivement.

    dire_bonjour()

dire_bonjour_nom("Toto")

reponse = test_temperature(14.5)
print(reponse)

    Erreurs fréquentes

    • Oubli des () en appelant la fonction : SyntaxError.
    • Mauvaise gestion des indentations : IndentationError.
    • Mauvaise gestion des paramètres donnés : TypeError.
    • Oubli de return : comportement anormal.
    • Confusion print/return dans une fonction : comportement anormal.
