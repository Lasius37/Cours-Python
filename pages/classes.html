    Créer une classe

    Une classe est un modèle qui permet de créer des objets (un peu comme un plan). On peut créer plusieurs objets de la même classe. Imaginons la classe Personnage. Chaque personnage a un nom, un âge, ... : ce sont ses attributs. Il peut aussi faire des choses : parler, ... : ce sont ses méthodes. Créer la classe Personnage équivaut à concevoir un plan pour créer n'importe quel personnage. Tous les personnages ainsi créés auront les mêmes attributs, avec des valeurs différentes, et les mêmes méthodes. On nomme une classe en PascalCase. 

    class Personnage:
    def __init__(self, nom, age):  # la méthode constructeur ; il faut lui donner deux paramètres
        self.nom = nom  # l'attribut nom ; on lui affecte la valeur du paramètre nom
        self.age = age  # l'attribut age ; on lui affecte la valeur du paramètre age

    def parler(self):  # la méthode parler
        print(f"Bonjour, je m'appelle {self.nom}")

    Créer une instance à partir d'une classe

    On peut créer un grand nombre d'objets à partir de la classe : on parle d'instances de la classe. Ici, perso_1 et perso_2 sont deux instances de la classe Personnage. Pour créer une instance, il faut passer une valeur par paramètre de la méthode __init__. 

    perso_1 = Personnage("Luke", 25)  # on passe des valeurs au paramètre du constructeur
perso_2 = Personnage("Yoda", 163)  # on crée une deuxième instance

    Utiliser les attributs d'une instance

    Chaque instance est accessible par son nom de variable (ici, perso_1 et perso_2). 

    print(perso_1.nom)  # affiche "Luke"
print(perso_2.nom)  # affiche "Yoda"
print(perso_1.age)  # affiche "25"
print(perso_2.age)  # affiche "163"

    Utiliser les méthodes d'une instance

    On peut considérer que les méthodes sont des fonctions, mais uniquement valables pour les instances de la classe. Toutes les méthodes prennent self comme premier paramètre. Dans le code, il n'est pas nécessaire de le fournir lors de l'appel d'une méthode. 

    perso_1.parler()  # affiche "Bonjour, je m'appelle Luke"
perso_2.parler()  # affiche "Bonjour, je m'appelle Yoda"

    La méthode spéciale constructeur : __init__

    Cette méthode est appelée au moment de l'initialisation de l'instance. C'est pour cela qu'il faut lui apporter les paramètres nécessaires pour qu'elle puisse affecter les valeurs voulues aux attributs de l'instance. Au moment de l'appel de cette méthode, les paramètres prennent les valeurs indiquées.

    La méthode spéciale d'optimisation de lisibilité : __str__

    Par défaut, un print() d'une instance affiche quelque chose de peu lisible.

    print(perso_1)  # affiche <__main__.Personnage object at 0x00000211175B6F90>

    Pour éviter cela, on crée la méthode __str__.

    class Personnage:
    def __init__(self, nom, age):
        ...

    def __str__(self):
        return f"{self.nom}, {self.age} ans."

    Ensuite, on peut refaire un print().

    print(perso_1)  # affiche "Luke, 25 ans"

    Erreurs fréquentes

    • Oublier self : TypeError.
    • Oublier les () avec une méthode : comportement anormal.
    • Confondre un attribut d'instance avec une variable "standard" : comportement anormal.
    • Mal orthographier __init__ ou __str__ : TypeError ou comportement anormal.

    Exercices

    1. Lire du code : qu'affiche le code suivant ?

      class Voiture:
    def __init__(self, marque, modele, annee):
        self.marque = marque
        self.modele = modele
        self.annee = annee

    def __str__(self):
        return f"{self.marque} {self.modele} - {self.annee}"

    def demarrer(self):
        print(f"{self.marque} {self.modele} démarre.")

v1 = Voiture("Renault", "Twingo", 2023)
print(v1)
v1.demarrer()

    def init(self, nom):

    def parler()
        print("Bonjour, je suis {nom}")

r = Robot("R2-D2")
r.parler
