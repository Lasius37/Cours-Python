    Créer une boucle for

    Cette boucle permet de répéter une série d'opérations un nombre déterminé de fois.

    for i in range(10):  # pour chaque nombre entre 0 et 9
    print(i * 2)  # on affiche le double du nombre
    print(i ** 2)  # on affiche le carré du nombre

    Créer une boucle while

    Cette boucle permet de répéter une série d'opérations tant qu'une condition est vérifiée.

    saisie_utilisateur = ""  # on crée une variable vide
while saisie_utilisateur != "oui":  # tant que la variable n'est pas égale à "oui"
    saisie_utilisateur = input("Tapez 'oui' en minuscules : ")  # on demande à l'utilisateur de taper "oui"

    Sortir d'une boucle while

    Dans certains cas, une boucle while peut être infinie. Pour en sortir, on utilise le mot break.

    while True:  # boucle infinie : True est toujours vrai...
    lettre = input("Taper une lettre : ")  # on demande une saisie à l'utilisateur
    if lettre == "a":  # on donne une condition
        print("OK !")  # on affiche un message de validation
        break  # on sort de la boucle si la condition est vraie
    else: # sinon...
        print("mauvais choix")  # on affiche un message d'erreur

    Erreurs fréquentes

    • Oubli des : à la fin de la ligne for ou while : SyntaxError.
    • Indentation incorrecte : IndentationError.
    • Boucle infinie involontaire (oublie de break) : le programme semble bloqué.
    • Mauvaise utilisation de range() : comportement anormal.
