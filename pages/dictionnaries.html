    Créer un dictionnaire

    • On peut créer un dictionnaire vide pour le compléter ensuite. On utilise les {}. 
      dictionnaire_vide = {}
    • On peut aussi créer un dictionnaire qui contient déjà des éléments : 
      eleve = {"nom": "Toto", "note": 14.5}
# clés : "nom" et "note", valeurs : "Toto" et 14.5

    Ajouter ou modifier un élément dans un dictionnaire

    La syntaxe est particulière : on donne la clé dans des [], et la valeur après le =. C'est la même syntaxe pour modifier une valeur dont la clé existe déjà : on écrase les données. 

    eleve["age"] = 12  # ici, l'âge est ajouté au dictionnaire
eleve["note"] = 15  # ici, la valeur de la note est remplacée

    Trouver un élément dans un dictionnaire

    On cherche la valeur en fonction de la clé.

    print(eleve["nom"])  # dans ce cas, on affichera "Toto"

    Supprimer un élément d'un dictionnaire

    On utilise le mot del.

    personnage = {
    "nom": "Dark Vador",
    "categorie": "Boss",
    "force": 18,
    "arme": "sabre laser"
}
del personnage["arme"]

    Parcourir un dictionnaire

    Il y a plusieurs manières de parcourir un dictionnaire avec une boucle for.

    for cle in dictionnaire.keys():
# parcourt le dictionnaire selon les clés

for valeur in dictionnaire.values():
# parcourt le dictionnaire selon les valeurs

for cle, valeur in dictionnaire.items():
# parcourt le dictionnaire selon les deux

    Erreurs fréquentes

    • Appel d'une clé inexistante : KeyError.
    • Confusion liste/dictionnaire : KeyError ou comportement anormal.
