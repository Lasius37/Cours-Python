    Créer un tuple

    Un tuple fonctionne comme une liste, mais une fois créé, il n'est pas modifiable. On peut l'écraser, mais pas modifier les valeurs qu'il contient. On utilise les ().

    coordonnees = (45, 32)

    Trouver un élément dans un tuple

    • Pour récupérer la position d'un élément dans le tuple (son index) : 
      semaine = ("lun", "mar", "mer", "jeu", "ven", "sam", "dim")
numero = semaine.index("jeu")
# les index commencent à 0, donc le jeudi est à l'index 3 : les index sont 0 ; 1 ; 2 ; 3 ; 4 ; 5 ; 6. Ici, numero prendra la valeur 3
    • Pour récupérer le 6ème élément dans le tuple 
      jour = semaine[5]
# les index commencent à 0. Ici, jour prendra la valeur "sam"

    Connaître la longueur d'un tuple

    semaine = ("lun", "mar", "mer", "jeu", "ven", "sam", "dim")
len(semaine)
# renvoie 7

    Parcourir un tuple

    On réalise une boucle sur le tuple, et chaque élément du tuple sera traité indépendamment.

    for jour in semaine:  # pour chaque élément du tuple "semaine"
    print(jour)

    Tuple ou liste ?

    Les tuples sont moins maniables (non modifiables), mais ils nécessitent moins de mémoire que les listes. Si vous voulez protéger les données en écriture, utilisez un tuple. Si vous voulez les rendre modifiables, utilisez une liste.

    Erreurs fréquentes

    • Tentative de modification d'un tuple : AttributeError.
    • Oubli des , de séparation : SyntaxError ou comportement anormal.
