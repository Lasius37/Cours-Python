    Déclarer une variable

    Les variables sont des boîtes, avec une étiquette (leur nom), et qui contiennent une donnée. On les nomme en snake_case.

    Dans les lignes ci-dessous, les textes en clair sont des commentaires.

    • Pour un nombre entier : 
      ma_variable_1 = 3
# la variable a pour valeur le nombre entier 3
    • Pour un nombre décimal : 
      ma_variable_2 = 3.14
# la variable a pour valeur le nombre 3.14
    • Pour une chaîne de caractères : 
      ma_variable_3 = "mon prénom"
# la variable a pour valeur le texte "mon prénom"
    • Pour un booléen : 
      ma_variable_4 = True
# la variable a pour valeur True

    Appeler une variable

    On utilise une variable créée plus haut.

    • On peut faire des calculs : 
      somme = ma_variable_1 + ma_variable_2
# calcul d'une somme, stockage dans une autre variable
    • On peut afficher un résultat : 
      print(somme)
# on affiche le résultat brut
    • On peut utiliser des chaînes de caractères formatées, ou f-string. Une f-string est une chaîne de caractères qui comporte une variable. 
      montant = 250
print(f"Votre compte est crédité de {montant} €")

    Principaux types de variables

    Les variables ont un type donné. Chaque type est utilisable d'une façon qui lui est propre.

    • Pour connaître le type d'une variable, on peut taper la commande : 
      print(type(ma_variable))
# on affiche le type de la variable
    • Un nombre entier est de type int.
    • Un nombre décimal est de type float.
    • Une chaîne de caractères est de type str.
    • Un booléen est de type bool. Il peut prendre la valeur True ou False.

    Erreurs fréquentes

    • Oubli des "" dans une chaîne de caractères : SyntaxError.
    • Oubli des () de print() : SyntaxError.
    • Oubli de f ou des {} dans une f-string : comportement anormal.
