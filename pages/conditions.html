    Tester une condition

    Un test de condition renvoie un booléen (True ou False).

    • Cas n° 1 : "si..." 
      if ma_variable == 0:  # si cette condition est vraie
    print("la variable est nulle")  # on affiche ce texte
    • Cas n° 2 : "si... sinon..." 
      if ma_variable > 0:  # si cette condition est vraie
    print("la variable est positive")  # on affiche ce texte
else:  # sinon
    print("la variable est négative ou nulle")  # on affiche ce texte
    • Cas n° 3 : "si... sinon si... sinon..." 
      if ma_variable < 0:  # si cette condition est vraie
    print("la variable est négative")  # on affiche ce texte
elif ma_variable > 0:  # sinon si cette autre condition est vraie
    print("La variable est positive")  # on affiche ce texte
else:  # sinon
    print("la variable est nulle")  # on affiche ce texte

    Opérateurs de comparaison

    == ... est égal à ...
    != ... est différent de ...
    < ... est strictement inférieur à ...
    > ... est strictement supérieur à ...
    <= ... est inférieur ou égal à ...
    >= ... est supérieur ou égal à ...

    Exemple :

    if ma_valeur >= 0:
#si la valeur de la variable est positive ou nulle

    Tester plusieurs conditions : les opérateurs logiques

    and ... et ... combine deux tests, les deux doivent être vrais
    or ... ou ... combine deux tests, un seul doit être vrai
    not ... non ... inverse un test
    • Utilisation de and : 
      if ma_valeur_1 >= 0 and ma_valeur_2 < 0:
# les deux conditions doivent être respectées
    • Utilisation de or : 
      if ma_valeur_3 == 0 or ma_valeur_4 < 0:
# une seule des deux conditions doit être respectée
    • Utilisation de not : 
      if utilisateur_connecte == False:
# syntaxe standard ; correcte, mais peu intuitive

if not utilisateur_connecte:
# syntaxe équivalente avec not, plus intuitive

    Chaîner des comparaisons

    if ma_valeur >= 0 and ma_valeur < 50:
 # syntaxe standard ; correcte, mais peu intuitive

if 0 <= ma_variable < 50:
 # syntaxe équivalente avec chaînage, plus facile à lire

    Erreurs fréquentes

    • Confusion de = avec == : SyntaxError.
    • Oubli des : après if, elif ou else : SyntaxError.
    • Oubli des indentations après if, elif ou else : IndentationError.
    • Condition impossible à atteindre : comportement anormal.
