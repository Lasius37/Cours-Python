    Calculer en Python

    Python est un excellent calculateur. Il réalise de nombreuses opérations mathématiques avec les symboles suivants (les priorités sont les mêmes qu'en mathématiques) :

    addition + 5 + 2 7
    soustraction - 5 - 2 3
    multiplication * 5 * 2 10
    division / 5 / 2 2.5
    division (quotient entier) // 5 // 2 2
    division (reste) % 5 % 2 1
    puissance ** 5 ** 2 25 
    print(5 + 2)
# Python affiche 7

    Récupérer une saisie utilisateur

    On peut récupérer des données pour établir un "dialogue" entre l'ordinateur et l'utilisateur. Attention, la variable sera toujours de type str. 

    age = input("Saisissez votre âge : ")
# le texte permet de guider l'utilisateur

    Changer le type d'une variable

    La fonction input() permet de récupérer une chaîne de caractères. Mais on peut avoir besoin de la convertir en nombre. 

    ma_variable_1 = int("32")  # renvoie le nombre entier 32
ma_variable_2 = float("37.5")  # renvoie le nombre décimal 37.5
ma_variable_3 = bool(1)  # renvoie le booléen True (pour 0, renvoie False)
ma_variable_4 = str(32)  # renvoie la chaîne de caractères "32"
ma_variable_5 = str(10.23)  # renvoie la chaîne de caractères "10.23"
ma_variable_6 = str(False)  # renvoie la chaîne de caractères "False"

    Erreurs fréquentes

    • Calcul avec des variables de types différents (ex : "Coucou" + 3 ) : TypeError.
    • Conversion d'une chaîne de caractères en nombre (ex: int("abc")) : ValueError.
    • Oubli du fait que input() renvoie une chaîne de caractères : TypeError.
